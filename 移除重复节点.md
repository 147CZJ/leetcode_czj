编写代码，移除未排序链表中的重复节点。保留最开始出现的节点。

示例1:

 输入：[1, 2, 3, 3, 2, 1]
 输出：[1, 2, 3]
示例2:

 输入：[1, 1, 1, 1, 2]
 输出：[1, 2]
```
 public ListNode removeDuplicateNodes(ListNode head) {
        HashSet<Integer>set = new HashSet<>();
        ListNode res  = head;
        while(head!=null&&head.next!=null){
            set.add(head.val);
            if(set.contains(head.next.val)){
                head.next = head.next.next;
            }
            else{
                head = head.next;
            }
        }
        return res;
    }
```
